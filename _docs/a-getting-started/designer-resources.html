Watch Face Quality Checklist

  1. All features that are included are working as intended (weather, location, time, step count, battery, etc)
  2. If your watch face includes the digital version of the time, both 24 and 12 hour time are included using the #Db# tag.
  3. Dim mode is present and functional
  4. Dim mode includes the time
  5. Watch face has both square and round backgrounds
  6. Your watch face description matches the watch face functionality accurately

Avatars

Watch face design

Good watch face design typically has a few characteristics, just like app design. They are focused, have a clear hierarchy and have great color harmony. Keep all of those in mind when designing.

Design specs

Canvas is 640 x 640px at 72 dpi/ppi. You might also want to consider that some watches have a chin at the bottom which would hide any design elements that would be down there on selected models. The editor also gives an option to preview the chin with the leftmost button, which is located right below canvas preview

Make sure you design for both square and circle.

You can do this by clicking advanced button on an image layer and then uploading different images for square and circle.
